# Personal Portfolio Website 🚀

A modern, responsive personal portfolio website built with React TS and FastAPI that is synced
and updated automatically with my PDF resume.

## Tech Stack 💻

- **Frontend:**
  - React
  - TailwindCSS
  - TypeScript

- **Backend:**
  - FastAPI
  - MongoDB
  - Python
 
Hosted with Vercel and Railway
